스프링은 객체지향적으로 잘 짜여진 프레임워크라고 합니다.  
그런 스프링 프레임워크를 이용하여 웹 개발과 OOP를 공부해보려고 합니다.

## 목표

스프링 프레임워크를 배우고 프레임워크를 사용하면서 객체지향에 대해서 알아봅니다.

## 스터디 방식

* 매주 정해진 시간(default: 토요일, 11시~ 6시)에 모여 각자 알아서 스터디 관련 책을 읽습니다.
* 정해진 진도 양이나 방식은 없습니다.
* 모르는 것이 있으면 서로 피드백을 해줍니다.

## 과정

1. 코드로 배우는 스프링 웹 프로젝트
  스프링의 기본에 대해서 알아보고 프로젝트를 만들어 본다
2. 토비의 스프링 3.1
  객체지향 관점에서 스프링에 대해서 자세히 알아본다
3. 토비의 스프링 5
  최신 스프링에 대해서 알아 본다
